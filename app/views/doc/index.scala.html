API Doc v1

Login

  • POST /auth/login

Logout

  • GET /auth/logout

Forgot Password

  • POST /auth/forgotPassword

Registration

  • POST /auth/register

User

  • GET /auth/user
  • PUT /auth/user/{id}
  • GET /auth/remove/{id}
  • PUT /auth/password
  • GET /auth/search?q={query string}

Feeds

  • GET /feeds
  • POST /feeds/create
  • GET /feeds/{id}
  • GET /feeds/{id}/timeline?page={page number}&pageSize={items per page}
  • GET /feeds/search?q={query string}
  • GET /feeds/popular
  • GET /feeds/popular/add/{id}
  • GET /feeds/popular/remove/{id}
  • GET /feeds/trending
  • GET /feeds/trending/add/{id}
  • GET /feeds/trending/remove/{id}
  • GET /feeds/nearby?radius={distance in kilometers}
  • GET /feeds/recent/{count}
  • GET /feeds/timeline?page={page number}&pageSize={items per page}
  • GET /feeds/remove/{id}
  • GET /feeds/follow/{id}
  • GET /feeds/unfollow/{id}

{
  username: "test@@email.com",
  password: "1111111"
